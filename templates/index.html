使用帮助

1. 点击 "Auth" 授权

2. 输入要导入对象的用户名，点击 "Get Repo" ，取得 Repository 列表

3. 默认所有 Repo 被选中，按需求取消选中，就可以排除该 Repo

4. 点击 "Run" ，开始导入，其实就是一个一个 Star ，成功的会变成绿色，失败的变成红色。
